# Image Generation Prompting Templates

## Midjourney Pattern
Short, descriptive phrases with style keywords and parameters at end.

### Basic Structure
```
[subject], [action/pose], [setting], [lighting], [style] --ar 16:9 --v 6
```

### Example
```
warrior princess, standing on cliff edge, stormy ocean background, dramatic golden hour lighting, oil painting style --ar 16:9 --style raw
```

### Style Keywords
- Lighting: golden hour, dramatic, soft, harsh, backlit, rim light
- Mood: ethereal, gritty, dreamy, cinematic, moody
- Medium: oil painting, watercolor, photograph, 3D render, digital art

## DALL-E Pattern
More natural language, descriptive sentences work well.

### Template
```
A [style] [medium] of [subject] [action] in [setting], [lighting description], [mood/atmosphere]
```

## Nano Banana Pro Pattern
6-factor structure: subject, composition, action, setting, lighting, style

### Template
```
[subject description], [camera angle/shot type], [action], [environment], [lighting quality], [visual style]
```

### Example
```
Samurai in red armor wearing a fierce oni mask, low angle shot, standing battle-ready, misty bamboo forest, dramatic side lighting, cinematic 4K
```
