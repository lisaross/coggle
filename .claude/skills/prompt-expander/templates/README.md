# Prompt Templates by Platform

Quick reference for platform-specific prompting patterns.

| Platform | Best For | Key Pattern |
|----------|----------|-------------|
| [Claude](claude.md) | Complex reasoning, coding | Role + Context + Task + Constraints |
| [Codex/GPT](codex.md) | Code generation, chat | Delimiters + Examples |
| [Gemini](gemini.md) | Multimodal, reasoning | Few-shot examples |
| [Image Gen](image-gen.md) | Midjourney, DALL-E, Nano Banana | Descriptive phrases + Parameters |
| [Video Gen](video-gen.md) | Sora, Runway, Kling | Storyboard + Camera direction |

## Usage
When expanding a prompt, detect the target platform and apply the appropriate template structure.
